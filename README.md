C5D Firmware
=========================
Production firmware for C5D

AUTHORS:  	
John & Nick @jdslabs
www.jdslabs.com

REQUIRES:	
Arduino Bootloader, Arduino 1.0.1

Usage
=========================

1) Place "boards.txt" file in your "arduino\hardware\arduino" folder.

2) Compile code and burn to C5 using any ISP compatible programmer.
